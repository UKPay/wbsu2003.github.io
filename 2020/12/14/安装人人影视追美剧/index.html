安装人人影视追美剧

发表于 2020-12-14 分类于 各种折腾 阅读次数： Valine：
本文字数： 1.3k 阅读时长 ≈ 1 分钟
人人影视客户端，是一个下载视频、追剧必备的利器！

人人影视客户端，是一个下载视频的利器，上面的的影视资源很多，而且还带搜索功能，可以直接搜索外部的资源。之前用过 windows 版本，今天我们研究下第三方基于官方 WEB 远程管理版封装的 docker 版本。

安装

在 docker 注册表中搜索 rrshare ，我选择了第六个 dreaminess/rrshareweb，之所以选择这个是因为更新日期比较近吧，因为网上的教程用的镜像已经是2年前的了，这个镜像只有一个版本，所以双击就可以下载。

卷

文件夹 装载路径
/your downloads /download

只需要将上面的 /your downloads 改为你自己的下载路径即可

端口

全部用的默认的端口

运行

在浏览器中输入 http://群晖:3001，就能看到主界面了

默认的 解锁密码 是 123456，可以在设置中进行修改；

存储路径 改为前面设置的 /download，如果不一致可能会出问题。

下载

在左侧 最新更新 里随便下了一个，速度可以达到 4~20MB/s

正常速度

最大峰值，但是并没有跑满我的带宽（理论上我的带宽应该要到 25MB/s ）

当然可以直接通过右上角的下载图标进行添加下载链接，不过支持的协议格式是人人影视自家的格式：yyets开头的。

追剧

去官网注册一个人人影视的账号，注册地址：http://www.rrys2020.com/

如果习惯用客户端，可以到下载页面：http://app.rrys.tv/，挑选适合自己平台的客户端。

因为 自动下载 默认勾选了 只下载订阅，所以在登录你的人人影视账号之后，所有被收藏的内容，当有最新发布时，都会被客户端自动下载。如果你喜欢美剧，用来追剧是不错的选择。

如果勾选了 全部下载，那么在 最新更新 里出现的都会被自动下载；

其他客户端

除了前面提到的客户端，还有一个值得关注的基于 Flutter 的客户端，这是个开源项目，其 Android 客户端支持边下边看的功能。
