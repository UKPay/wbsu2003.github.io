Web资源文件管理器KodExplorer

发表于 2021-05-03 分类于 各种折腾 Waline： 阅读次数： 本文字数： 1.3k 阅读时长 ≈ 1 分钟
KodExplorer是一款快捷高效的私有云和在线文档管理系统

前言

在还没有接触 NAS 的时候，一直使用一台旧电脑来完成现在 NAS 承担的功能，比如用 Plex 做媒体服务器，用 TwonkyMedia server 或者 FreeMi UPnP Media Server 做 UPnP 服务器，用 Daemon Sync 做手机照片同步服务器，用 Seafile 做网盘等等，而 KodExplorer 则满足了我基于 Web 方式远程管理 Windows 主机的需求。

什么是 KodExplorer？

KodExplorer 是一款快捷高效的私有云和在线文档管理系统，为个人网站、企业私有云部署、网络存储、在线文档管理、在线办公等提供安全可控，简便易用、可高度定制的私有云产品。采用 windows 风格界面、操作习惯，无需适应即可快速上手，支持几百种常用文件格式的在线预览，可扩展易定制。

什么是KodBox？

KodBox 是在 KodExplorer 基础上进行了系统重构的全新产品。 为满足系统更强性能、更安全、更多特性的拓展需求，Kodbox 对底层架构、存储方式、权限机制等进行了重构，同时继承并升级了 KodExplorer 优秀前端体验。

安装

在群晖上以 Docker 方式安装。

在注册表中搜索 kodexplorer ，选择第二个 baiyuetribe/kodexplorer，版本选择 latest。

卷

在 docker 文件夹中，创建一个新文件夹，并将其命名为 kod。

文件夹 装载路径
docker/kod/ /code

端口

本地端口 容器端口
7480 80

运行

在浏览器中输入 http://群晖IP:7480，就能看到主界面了，第一次需要设置 admin 的密码，显然 KodExplorer 是支持多用户管理的

设置完成后需要登录

默认界面很像 Windows 的资源管理器

除了文件管理外，还可以上传下载

还可以通过安装插件扩展各种文件打开方式，从而实现在线预览，支持的格式相当的多，比如视频

播放效果相当流畅

当你返回桌面后，会发现很像 Web 版的 windows 系统，尤其是在选择了合适的主题之后

KodExplorer 解决了文件在线存储与管理、共享和跨平台访问、在线办公影音娱乐等一系列问题，使得远程可以拥有本地操作一样方便、快捷、安全的体验。

其他

除了 Docker 方式，群晖上还可以从套件中心安装

只不过还需要安装依赖的套件： WebStation 套件 + Apache2.2 套件 + PHP7.0 套件，这是 KodExplorer 官方推荐的组合，相对 Docker 安装方式略显复杂。
