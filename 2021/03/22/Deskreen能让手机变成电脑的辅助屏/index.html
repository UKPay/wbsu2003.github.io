Deskreen能让手机变成电脑的辅助屏

发表于 2021-03-22 更新于 2021-03-24 分类于 各种折腾 Waline： 阅读次数： 本文字数： 2.7k 阅读时长 ≈ 2 分钟
能将任何带有 Web 浏览器的设备转换为电脑的辅助屏幕。


（题图来自官网）

什么是 Deskreen ？

  • Deskreen 是一个 electron.js 应用程序，通过 WebRTC 将您的桌面实时流传输到任何设备上的 Web 浏览器。
  • Deskreen 是一款桌面应用程序，可以通过 WiFi 将任何带有网络浏览器的设备变成电脑的辅助屏幕。
  • Deskreen 可用于将整个计算机显示镜像到任何具有 Web 浏览器的设备屏幕上。您还可以限制 Deskreen 只选择一个要共享的应用程序窗口视图，这对于演示非常有用。
  • Deskreen 的最大特点是可以使用任何设备作为辅助屏幕。
  • Deskreen 是开源的，源代码地址（https://github.com/pavlobu/deskreen）

前言

Deskreen 作为一款桌面应用程序，支持 Windows, Linux, MacOS 系统，对存储、内存和 CPU 的要求都不高

  • CPU (Windows)： 任何现代双核 CPU、较弱的 CPU 在 Windows 上的当前版本都可能出现性能问题。我们将来会把它修好的。

  • CPU (MacOS)： 任何现代 CPU

  • CPU (Linux)： 任何现代 CPU，甚至支持单核 ARM，例如 Raspberry Pi

如何使用

为了研究 Deskreen 的用法，准备拿 Windows 10 台式机和 Android 手机做测试。虽然台式机已经比较老旧了，但性能肯定能满足要求。

如果您的相机无法识别小的二维码 (QR)，请单击二维码 (QR) 将其放大

高级用法

如果只是我们前面介绍的用法，最多也就是复制屏的用法，而如果要变成扩展屏，按官方的说法需要用到 虚拟显示适配器 。

所谓 虚拟显示适配器 实际上就是欺骗主机，让主机以为存在第二个显示器，这玩意儿可以是硬件，也可以是软件。

硬件方案

某宝上有很多种不同的叫法，比如：

  • 锁屏宝
  • 虚拟显示器
  • 显卡欺骗器
  • EDID 仿真器
  • EDID 模拟器
  • 假负载

接口支持 VGA 、HDMI 、DP 、DVI 、Type-C 等，分辨率一般是 1080P 和 4K，价格从几块到几十块，可以根据自己的需要选择。

软件方案

这个方案简单而免费，可以在您的系统上创建一个软件监视器，但是这个解决方案仅适用于 Windows 10。

安装

  • 下载虚拟显示驱动程序 https://www.amyuni.com/downloads/usbmmidd.zip 或者 usbmmidd.zip
  • 将zip文件解压缩到一个空文件夹，例如 C:\temp\usbmmidd
  • 以管理员身份打开命令提示符窗口（除非您以 以管理员身份运行，否则无法将设备添加到系统中）
  • 运行下面的命令
    • cd C:\temp\usbmmidd（或您选择的任何目标文件夹）
    • deviceinstaller64 install usbmmidd.inf usbmmidd
    • 如果显示 Drivers installed successfully.，就执行下一条命令
    • deviceinstaller64 enableidd 1
  • 如果您使用的是32位系统，请用 deviceinstaller 替换 deviceinstaller64
  • 更简单的办法，请右键单击 usbmmidd.bat 并 以管理员身份运行。
