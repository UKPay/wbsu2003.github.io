电影管理工具Radarr

发表于 2021-03-08 分类于 各种折腾 Waline： 阅读次数： 本文字数： 1.2k 阅读时长 ≈ 1 分钟
Radarr 是 Sonarr 的分支，去掉了剧集功能，添加了电影下载功能。

什么是Radarr？

Radarr 是 Sonarr 的分支，去掉了剧集功能，添加了电影下载功能，可以监控多个 RSS 源，配合 BT 下载客户端自动下载，也可以手动搜索。

安装

在群晖上以 Docker 方式安装。

在注册表中搜索 radarr ，选择第一个 linuxserver/radarr，版本选择 latest。

卷

在 docker 文件夹中，创建一个新文件夹，并将其命名为 radarr，然后在 radarr 文件夹中再创建一个新的子文件夹命名为 config 。

文件夹 装载路径 说明
docker/radarr/config /config 数据库和radarr设置
path/to/tvseries /movies 电影库的位置
path/to/downloadclient-downloads /downloads 下载管理器下载目录的位置，我指向了 qBittorrent 的下载目录

端口

直接用了默认的端口 7878

本地端口 容器端口
7878 7878

环境

可变 值
TZ 设置为 Asia/Shanghai
PGID <gid> 设为了 1000
PUID <uid> 设为了 1000

设置

在浏览器中输入 http://群晖IP:7878，就能看到主界面了

设置语言

Setting –> UI –> Language

点 Save Changes 保存后，刷新一下页面

设置下载客户端

设置 –> 下载客户端 –> ＋ –> qBittorent

选择 qBittorrent

填完之后，点 测试，如果正常，测试按钮会变成绿色的勾

如果失败，测试按钮会变成红色的感叹号

保存 之后就可以了

添加搜刮器

设置 –> 搜刮器 –> ＋ –> Torznab –> `Custom``

在 Jackett 界面，点击 Copy Torznab Feed

粘贴到 URL，API Key 在 Jackett 界面右上角

测试一下，如果没问题就 保存

使用

电影 –> 添加新电影

可以直接支持查询 电影名称，tmdb 和 imdb

搜索 姜子牙

添加电影

和 Sonarr 一样，不会对过去已播出的电影自动下载

可以手动下载

切换到 qBittorrent 就可以看到刚加入下载的文件

除了自己搜索订阅外，在 发现 中还会有主动的推送
