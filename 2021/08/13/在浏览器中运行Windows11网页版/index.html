在浏览器中运行Windows11网页版

发表于 2021-08-13 分类于 各种折腾 Waline： 阅读次数： 本文字数： 1.4k 阅读时长 ≈ 1 分钟
开源项目 Windows 11 in React 旨在使用 React、CSS (SCSS) 和 JS 等标准 Web 技术在 Web 上复刻 Windows 11 桌面体验。

按照微软的预告，Windows 11 正式版会在 2021 年底推出。现在网上已经有了 Windows 11 ISO 预览版本，不过应该还是开发测试版，并不能应用于正式环境。

老苏第一时间就下载了官方的升级检测工具，毕竟能直接运行 Android 还是很吸引人的。

官方检测工具

  1. 下载 Windows PC 健康检查应用程序

因为过高的配置要求导致微软被狂喷，目前这个工具已经被微软删除了，不过还可以在第三方下载，地址老苏就不贴了，搜 Windows PC 健康检查应用程序 就能找到

  1. 安装并运行

  1. 开始检查

  1. 最低要求

让老苏失望的是，老苏的台式机也不满足 Windows 11 的安装要求，大部分机器不能升级的原因是不支持 TPM 2.0，但老苏的机器显然不是，检测工具几乎没有提供关于如何修复任何问题的信息和指导，所以老苏又下载了第三方的检测工具。

第三方检测工具

WhyNotWin11 是个开源项目：https://github.com/rcmaehl/WhyNotWin11，可以在这里下到最新的版本：https://github.com/rcmaehl/WhyNotWin11/releases

老苏直接下了最新的 v2.3.2，文件很小

直接运行，如果报有毒信任一下即可

从测试结果看应该是 CPU 不能满足要求，查了下最低需要 英特尔 8 代 CPU，看来注定是与 Windows 11 无缘了。

Web 版 Windows 11

虽然 Windows 11 用不了，但是国外有个大牛在 Web 上复刻了 Windows 11 的界面及部分功能，可用于体验 Windows 11 的一些功能。

什么是 Windows 11 in React ？

这个开源项目旨在使用 React、CSS (SCSS) 和 JS 等标准 Web 技术在 Web 上复刻 Windows 11 桌面体验。目前该项目已经实现部分功能，例如微软商店、Edge 浏览器等。

可以直接在线体验： https://win11.blueedge.me

不过可能受网络影响，并不是特别顺畅，所以老苏打算在本地安装
