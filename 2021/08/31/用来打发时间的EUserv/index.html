用来打发时间的EUserv

发表于 2021-08-31 更新于 2021-09-01 分类于 各种折腾 Waline： 阅读次数： 本文字数： 2.1k 阅读时长 ≈ 2 分钟
EUserv 是只有 IPv6 地址的免费 VPS 。

什么是 EUserv ？

EUserv 是一家德国主机商，提供专用服务器托管，虚拟专用服务器，云服务，网站托管和域注册服务。EUserv 默认提供的免费 VPS 仅提供 IPv6 地址，支持按月续期。

EUserv 其实是支持 IPv4 的，只不过需要花钱而已

什么是 VPS ？

VPS 服务器（虚拟专用服务器）（Virtual Private Server，或简称 VPS）是利用虚拟服务器软件（如微软的 Virtual Server、VMware 的ESX server、SWsoft 的 Virtuozzo ）在一台物理服务器上创建多个相互隔离的小服务器。【百度百科】

申请

怎么申请 EUserv 的网上教程太多了，这里就略过了，老苏从申请到开通足足用了 8 天，在自己都快不记得的时候，终于收到了邮件

使用

登录后台，首先要找到 IP 和密码，这个在 Serverdata 中，一定要做好记录

IPv6 地址不要带后面的 /128

对于这种纯 IPv6 的 VPS ，怎么连上去设置是个大问题，老苏研究了很多网上的教程，大致做个总结：

  • 第一种是在 Windows 10 下，通过设置 Teredo 服务器来开启 IPv6 访问，这种方法一旦搞定可以一劳永逸，可惜不知道是老苏太笨，还是教程过期，总之没搞成功，虽然 ipconfig /all 中已经看到分配的 IPv6 地址

  • 第二种是正好你的宽带已经支持 IPv6 ，但是老苏所在的区域还不支持

测试地址：https://test-ipv6.com/index.html.zh_CN

  • 第三种方式是通过中转网站，比如 嘿哟终端 ：（https://heyterm.com）来访问，但是老苏试过所以的节点，不是超时就是无法连接，估摸着用的人太多了吧

  • 第四种方式和第三种有点类似，但需要你自己有一台同时支持 IPv4 和 IPv6 的 VPS 来做 SSH 中转

  • 第五种用手机做热点，一般来说手机拨号默认采用的 APN 协议同时支持 IPv4/IPv6，如果不是自己改一下就行

所以对老苏来说，可行的方案只有第五种了，但是不是每次都需要这么麻烦而且不经济呢？显然不是，因为老苏想到了 Tailscale

相关文章：『 外网访问群晖的新方案Tailscale 』

Tailscale 本身是完全支持 IPv6 的，即使您的 ISP 不支持 IPv6，Tailscale 也可以将 IPv6 数据包加密后通过 WireGuard 隧道在 Tailscale 节点之间传输，更多关于 Tailscale IPv6 的相关内容请参考：https://tailscale.com/kb/1121/ipv6

所以 EUserv 到手之后的第一件事情就是 SSH 登录并安装 Tailscale，之后除非重装系统，否则你都可以用 Tailscale 的 IPv4 地址来登录了

安装教程可以在 https://tailscale.com/download/linux/centos-7 找到，应该基本覆盖上了 EUserv 支持的系统

接下来就和一般的 IPv4 的 VPS 一样使用了，你可以安装宝塔，或者直接搭建 LNMP，只要性能允许。

当然如果你搭的网站想要给别人用，还需要再套一个 Cloudflare，不然别人还是访问不了，这方面的教程也很多，有兴趣可以自己研究。

续期

可能是因为用的人太多，EUserv 已经改成了需要每个月手动续期，当然这种小事情肯定难不倒聪明的网友，有利用 github action 自动续期的，也有利用 腾讯云函数 自动续期的，不过老苏注意到，有人因此被封了 github 账号。

小结

EUserv 有很多优点，比如不光免费还不要用信用卡认证，所以用来学习 linux、搭建网站是不错的选择

但是说实话，性子急的人不适合玩 EUserv ，不光光是因为申请时间长，而是 EUserv 中途经常会玩失联，而且一旦失联时间还不好说，快则几个小时，慢则几天，期间能证明它还存在的唯一证据只有在 Tailscale 的管理页面上还显示着 Connected

上一次的 Reinstallation 和 Webreboot 过去了四天还没处理

但是它居然活过来了，现在宝塔里的状态
